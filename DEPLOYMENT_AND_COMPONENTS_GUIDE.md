## QUICK SETUP

### Option 1: Clone & Build Locally

```bash
clone https://github.com/shreepranavg2014-a11y/fitness-tracker.git
cd fitness-tracker
npm install
ng serve
```

### Option 2: Deploy to Netlify (EASIEST)

1. Push remaining component files to GitHub
2. Build: `ng build --configuration production`
3. Deploy `dist/fitness-tracker` to Netlify
